# 简介
替换项接口是最常用的接口。在脚本项目中调用`find_and_replace_in_paragraph(paragraph, old_text, new_text)`





# 操作方式
### Step0:
修改skill目录下的`project_template.json`，添加对应的替换项。
![](替换项接口/image.png)
需要根据doc里边原文内容确定`replacement_pattern`，然后填入json模板。添加后即可由脚本自动完成替换工作。
注意：
默认的替换范围是全文：包括所有的section中的页眉、页脚、段落、表格。
### Step1（定制时）:
在`doc_fill.py`脚本中找到对应的替换代码逻辑，进行定制。

![](替换项接口/image-1.png)
