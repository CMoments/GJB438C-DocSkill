### 3.3 find_and_rep.py - 查找替换模块

实现格式保持的文本替换。

```python
def find_and_replace_in_paragraph(paragraph, old_text, new_text):
    """
    在段落中查找并替换文本,保持格式。

    处理两种情况:
    1. 文本在单个 run 中:直接替换
    2. 文本跨越多个 run:清空所有 run,在第一个 run 中放入替换后文本
    """
    if old_text not in paragraph.text:
        return False

    # 尝试在单个 run 中替换
    for run in paragraph.runs:
        if old_text in run.text:
            run.text = run.text.replace(old_text, new_text)
            return True

    # 处理跨 run 情况
    full_text = paragraph.text
    if old_text in full_text:
        for run in paragraph.runs:
            run.text = ""
        if paragraph.runs:
            paragraph.runs[0].text = full_text.replace(old_text, new_text)
        return True

    return False
```

**注意事项**:
- Word 将文本分割成多个 run 以支持不同格式
- 简单替换可能破坏格式
- 跨 run 替换会丢失中间格式

## 6. 测试

### 6.1 单元测试

```python
# tests/test_find_and_rep.py
import pytest
from docx import Document
from scripts.main.find_and_rep import find_and_replace_in_paragraph

def test_simple_replacement():
    """测试简单替换"""
    doc = Document()
    para = doc.add_paragraph("Hello World")
    result = find_and_replace_in_paragraph(para, "World", "Python")
    assert result == True
    assert para.text == "Hello Python"

def test_cross_run_replacement():
    """测试跨 run 替换"""
    doc = Document()
    para = doc.add_paragraph()
    para.add_run("Hello ")
    para.add_run("World")
    result = find_and_replace_in_paragraph(para, "Hello World", "Hi Python")
    assert result == True
    assert para.text == "Hi Python"
```
