# word-fillter-438C 说明文档

> 基于 GJB 438C 标准的软件需求规格说明书自动生成工具

## 项目简介

**word-fillter-438C** 是一个 Claude Code Skill,用于自动生成符合 GJB 438C 标准的软件需求规格说明书（Software Requirements Specification, SRS）。通过交互式问答收集需求信息,结合预设模板和替换模式,快速生成规范化的 SRS 文档。

### 核心特性

- **标准合规**: 严格遵循 GJB 438C 军用软件文档标准
- **交互式填写**: 通过 AskUserQuestion 工具引导用户完成需求收集
- **模板驱动**: 基于 JSON 模板的结构化配置管理
- **自动验证**: 内置验证机制确保文档质量
- **格式保持**: 在替换内容时保留 Word 文档原始格式

## 快速开始

### 前置条件

- Python 3.8+
- python-docx 库
- Claude Code CLI

### 安装

```bash
# 克隆或下载项目
git clone <repository-url> word-fillter-438C
cd word-fillter-438C

# 安装依赖
pip install python-docx
```

### 使用方式

1. **启动 Skill**: 在 Claude Code 中调用此技能
2. **填写项目信息**: 通过问答完成 `config.json` 的 project 部分
3. **填写文档信息**: 继续问答完成 `config.json` 的 document 部分
4. **填写需求信息**: 通过问答完成 `project.json`
5. **生成文档**: 执行处理脚本生成最终 SRS 文档
6. **验证输出**: 检查文档完整性

## 配置文件说明

### config.json 结构

```json
{
  "project": {
    "overview": { "type": "text_generation", "content": "..." },
    "scope": { "type": "text_generation", "content": "..." },
    "background": { "type": "text_generation", "content": "..." },
    "objectives": { "type": "text_generation", "content": "..." },
    "features": { "type": "text_generation", "content": "..." },
    "target_users": { "type": "text_generation", "content": "..." },
    "constraints": { "type": "text_generation", "content": "..." },
    "assumptions": { "type": "text_generation", "content": "..." },
    "references": { "type": "text_generation", "content": "..." },
    "terminology": { "type": "text_generation", "content": "..." }
  },
  "document": {
    "密级": { "type": "original", "replacement_pattern": "内部", "content": "..." },
    "阶段": { "type": "original", "replacement_pattern": "阶段： ", "content": "..." },
    "版次": { "type": "original", "replacement_pattern": "A 版", "content": "..." },
    "外部型号+产品名称": { "type": "original", ... },
    "产品型号": { "type": "original", ... },
    "公司名称": { "type": "original", ... },
    "日期": { "type": "original", ... }
  }
}
```

### project.json 结构

```json
{
  "template_info": {
    "name": "GJB 438C 软件需求规格说明书",
    "file": "[08]软件需求规格说明-438C.doc",
    "encoding": "gbk"
  },
  "structure": {
    "1": { "title": "范围", "placeholders": [...] },
    "2": { "title": "引用文档", "type": "table" },
    "3": { "title": "需求", "placeholders": [...] },
    "4": { "title": "合格性规定", "type": "table" },
    "5": { "title": "需求可追踪性", "type": "table" },
    "6": { "title": "注释", "placeholders": [...] }
  }
}
```

## 字段类型说明

| 类型 | 说明 | 处理方式 |
|------|------|----------|
| `original` | 原始文本 | 直接使用用户输入 |
| `text_generation` | 文本生成 | 根据描述生成文本 |
| `content_generation` | 内容生成 | 根据提示生成结构化内容 |

## 命令行用法

```bash
python main.py \
  --template "[08]软件需求规格说明-438C.docx" \
  --config "config.json" \
  --project "project.json" \
  --output "output/[08]软件需求规格说明-438C-filled.docx"
```

### 参数说明

| 参数 | 缩写 | 说明 | 必需 |
|------|------|------|------|
| `--template` | `-t` | Word 模板文件路径 | 是 |
| `--config` | `-c` | config.json 文件路径 | 是 |
| `--project` | `-p` | project.json 文件路径 | 是 |
| `--output` | `-o` | 输出文件路径 | 是 |
| `--help` | `-h` | 显示帮助信息 | 否 |
