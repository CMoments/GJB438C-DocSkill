# GJB 438C 模版目录说明

截至本目录维护时，**符合 GJB 438C-2021 的现成模板暂未找到**。

关于新标准，基本沿用旧标准，进行了部分增删。

到 `GJB 438C-2021 -word`文档中第五节规定了新标准的文档，在附录 A–T可以找到每类文档的"正文格式"，但是只有正文格式，符合新标准的模板暂未找到，欢迎大家提供。



本目录当前包含以下资料：

| 路径 | 说明 |
|------|------|
| `GJB 438C-2021 -word版本 .doc` | GJB 438C-2021 标准原文（Word 版） |
| `438B-2009全套模板/` | 旧标准 GJB 438B-2009 的 31 类模板集（仅供参考） |
| `438C-2021全套模板/` | 本项目按附录 A–T 搭建的全套 20 份 438C-2021 模板 |



## 关于"符合 438C-2021 的模板"

在 `GJB 438C-2021` 标准中：

- **第五节"详细要求"** 规定了新标准的文档目录（5.1–5.20，共 20 类文档）
- **附录 A–T** 给出每类文档的"**正文格式**"

但是附录提供的**只有正文格式说明，并非可直接使用的模板文件**。官方尚未发布配套的 `.docx` 模板。本目录的 `438C-2021全套模板/` 即是本项目根据附录描述搭建的 20 份模板。

1. **直接使用**：从 `438C-2021全套模板/` 取用所需模板，按项目情况填写。

3. **欢迎贡献**：如有更完善的合规模板（例如补充示例文本、完善封面表格、修正章节结构），欢迎提 PR 补入本仓库。

## 与 438B-2009 的主要差异（影响模板结构）

| 变化类型 | 涉及文档 |
|----------|----------|
| **已删除** | 软件配置管理计划、软件质量保证计划、软件输入/输出手册、软件中心操作员手册、计算机操作手册、软件配置管理报告、软件质量保证报告、软件研制任务书 |
| **合并** | 软件概要设计说明 + 软件详细设计说明 → **5.11 软件设计说明（SDD）** |
| **新增/重点修订** | 软件测试计划、运行方案说明、系统/子系统规格说明、接口需求规格说明、系统/子系统设计说明、接口设计说明、数据库设计说明、软件研制总结报告 |

因此 `438B-2009全套模板/` 里的模板**不能直接作为 438C-2021 的合规模板使用**，仅可作为旧标准的历史参考。当然也有部分单位沿用旧模板。
